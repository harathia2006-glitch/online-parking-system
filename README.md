Smart Parking Management System

Project Description:
Smart Parking Management System is a web-based application developed using Python Flask and SQLite. It helps users book parking slots online and allows administrators to monitor available and occupied parking spaces.

Technologies Used:

Frontend: HTML, CSS
Backend: Python Flask
Database: SQLite

Main Modules:

Booking Module – Users enter details like name, vehicle number, vehicle type, date, and timings to book a parking slot.
Slot Generation – System automatically assigns a parking slot number.
Database Module – Stores booking details securely.
Dashboard Module – Displays total slots, occupied slots, available slots, and booking information.

Advantages:

Saves time in finding parking.
Reduces manual work.
Easy parking management.
Maintains digital records.

Applications:

Shopping malls
Hospitals
Airports
Colleges

Future Enhancements:
